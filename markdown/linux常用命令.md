# 软件安装方面

+ sudo apt-get  update
+ sudo apt-get uprdge

+ sudo apt install ..deb 安装软件
+ sudo dpkg -i ...deb
+ mkdir hello 创建文件夹
+ touch hello.txt 创建文件
+ sudo apt purge netease-cloud-music 卸载软件

+ snap list列出snap包
+ dpkg --list 列出所有安装的包
